# Apa itu Tailwind CSS?

Sebuah framework CSS yang mengedepankan **Utility-first** berbeda dengan framework pada umumnya yang mengedepankan **component-first**.

## Utilities

- Preflight
- Layout & Positioning
- Rounding & Shadow
- Pseudo Classes
- Flexbox & Grid
- Spacing & Sizing
- Typhography
- Background & Borders
- Effect & Filters
- Transition, Transformation, & Animation
- Tables
- Color

## Features

- Interactivity
- Responsive Design
- Dark Mode
- Reusability
- Custom Styles
- Function & Directives
